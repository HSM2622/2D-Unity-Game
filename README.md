# 🚀 2D Warrior

## 📄 プロジェクトの概要 (Project Overview)

障害物とモンスターが混在するステージを通過し、ボスを倒して勝利を目指す **2D横スクロールアクションアドベンチャーゲーム**。

## 🛠️ プロジェクトの状態 (Project Status)

**デモ版 (Demo Version)**
* **実装内容:** 障害物ステージ 2つ、ボスステージ 1つを実装済み。
* **目的:** ポートフォリオ展示用として、コアな戦闘システムとレベルデザインの能力を示すことを目的としました。

---

## 🖥️ インストールと実行方法 (Setup & Run)

ローカル環境でプロジェクトを開くための手順です。

### 1. 動作環境

| 項目 (Item) | 詳細 (Detail) |
| :--- | :--- |
| **Unity バージョン** | **Unity 2021.3.7f1** |
| **動作確認 OS** | Windows / macOS |

### 2. 実行手順

1.  GitHubからリポジトリをクローンします。
    ```bash
    git clone [ここにGitHubリポジトリのURLを挿入]
    ```
2.  Unity Hubを開き、プロジェクトを登録します。
3.  `Assets/Scenes/MainScene.unity` ファイルをダブルクリックして開きます。
4.  Unityエディタの **'Play'** ボタンを押してゲームを開始します。

## ⚙️ 技術スタックと使用アセット (Technology Stack & Assets)

プロジェクトの実装に使用した主な技術と外部アセットの情報です。

### 1. 使用技術

* **Unity Engine:** **2021.3.7f1**
* **言語:** C#
* **コアシステム:** **リアルタイム戦闘**、2Dプラットフォーム制御、簡易的な敵AI

### 2. 主要アセットとフォルダ構造

以下の外部アセットと主要フォルダを使用してプロジェクトを管理しています。

* **アセット (外部ライブラリ):**
    * **TextMesh Pro:** UIテキスト表示に使用。
* **アートアセット (グラフィック):**
    * **Bringer Of Death**
    * **Monsters Creatures Fantasy**
    * **Simple 2D Platformer BE2**
    * **Warrior free set**
* **その他主要フォルダ:**
    * **Scripts:** 全ての C# スクリプトを管理。
    * **TileMap:** レベルデザインに使用したタイルマップデータ。
    * **UI:** ユーザーインターフェース関連のプレハブおよび画像。
